Building the example scene for Android:

  1. From “Build Settings”, under “Platform”, choose Android and click on “Switch Platform”.
  2. Go to Player Settings → Other Settings and make sure you change the “Scripting Backend” to “IL2CPP” and include both the “ARMv7” and “ARM64” Target Architectures.
  3. Go to Player Settings → Publishing Settings and check the Custom Main Manifest option. This will generate a custom manifest file that you can find in Assets → Plugins → Android. Open this file and add a new , add the following four lines inside, and save the file:
