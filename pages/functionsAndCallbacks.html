Available Functions & Callbacks:

Tamatem Plus SDK allows you to connect your users with the Tamatem ID through our public functions. The following functions can be used by accessing the singleton instance “TamatemConnectPlugin.Instance”:

  1. Connect: Call TamatemConnectPlugin.Instance.Connect() to connect the user to Tamatem Plus. You can use the callback function TamatemConnectPlugin.Instance.OnUserConnected. You can also use the callback function TamatemConnectPlugin.Instance.OnRedirectedToApp to place any logic needed when the user is redirected to the app during the connection process.
  2. GetConnectionStatus: Call TamatemConnectPlugin.Instance.GetConnectionStatus() to check if the user is connected or not. You can use the callback function TamatemConnectPlugin.Instance.OnGetConnectionStatus.
  3. GetInventoryItems: Call TamatemConnectPlugin.Instance.GetInventoryItems(string isRedeemed) to get the items purchased by the user on Tamatem Plus. Send “true” to get the redeemed items, “false” to get the unredeemed items and “all” to get all the items. You can use the callback function TamatemConnectPlugin.Instance.OnGetInventoryItems.
  4. . RedeemAllItems: Call TamatemConnectPlugin.Instance.RedeemAllItems() to redeem all items. You can use the callback function TamatemConnectPlugin.Instance.OnAllItemsRedeemed.
  5. GetUserInfo: Call TamatemConnectPlugin.Instance.GetUserInfo() to get the user data from Tamatem Plus. You can use the callback function TamatemConnectPlugin.Instance.OnGetUserInfo.
  6. Logout: Call TamatemConnectPlugin.Instance.Logout() to clear the Tamatem user session and clear the stored data on the device (Please note that this will NOT delete the stored data from Tamatem Plus records). You can use the callback function TamatemConnectPlugin.Instance.OnLogout.
You can find an example script to check out how these functions are used. To access this script, please go to Assets → TamatemConnect → Examples→ ExampleScripts → TamatemConnectExampleScript.
