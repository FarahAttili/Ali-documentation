Where to get the SDK?

Currently, you can get it from the following GitHub link (Please make sure to always use the latest version): GitHub - Tamatem-Plus/Unity-SDK

Supported Versions

Tamatem Plus Unity SDK works with the following Unity Engine versions: 2022 2021 2020 2019

Please make sure you are using a “Long Time Support” (LTS) version of Unity.
